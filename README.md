## Запуск фронтенда

1. **Скачайте репозиторий**

2.  **Перейдите в frontend**:
     ```bash
     cd TravelAgency/frontend
     ```
3. **Установите все для работы**:
   ```bash
     npm install
     ```
4. **Запустите проект**:
    ```bash
     npm run dev
     ```
## Запуск Бекенда

1.  **Запуск бекенда**:
     ```bash
     python -m venv venv ./venv/Scripts/activate.bat pip install -r requirements.txt python main.py
     ```
